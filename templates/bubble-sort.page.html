What is Bubble Sort?

Bubble Sort is a basic sorting algorithm that repeatedly steps through the list of elements, compares adjacent elements, and swaps them if they are in the wrong order. The process continues until list is sorted. This algorithm gets its name because smaller elements "bubble" to the top of the list like air bubbles in water.

How Bubble Sort Works?

Bubble Sort algorithm works same as air bubble works in water. After each steps, smaller elements bubbles towards top of the list. Step by Step breakdown of bubble sort is as follows.

  1. Start at the beginning of the list.
  2. Compare the first two elements (element at index 0 and element at index 1).
  3. If the first element is greater than the second element, swap them.
  4. Move to the next pair of elements (element at index 1 and element at index 2), and repeat the comparison and swapping if necessary.
  5. Continue this process until you reach the end of the list.
  6. After one complete pass through the list, the largest element will have "bubbled up" to the last position.
  7. Repeat steps 1-6 for the remaining unsorted elements (excluding the last element, which is already sorted).
  8. Continue this process until no more swaps are needed in a pass, indicating that the list is fully sorted.

Bubble Sort Implementation in C++

Bubble Sort implementation in C++ is as follows

          #include<iostream>using namespace std;void bubblesort(int arr[], int n) {  for(int i=0; i<n - 1; i++) {    for(int j=0; j<n - 1 - i; j++) {      if(arr[j] > arr[j + 1]) {        int temp = arr[j];        arr[j] = arr[j + 1];        arr[j + 1] = temp;      }    }  }}int main() {  int arr[] = {4, 5, 3, 1, 6, 2};  int pos[] = {0, 50, 100, 150, 200, 250};  int n = sizeof(arr)/sizeof(arr[0]);    bubblesort(arr, n);    cout << "array after sorting is " << endl;    for(int i=0; i<n; i++) {    cout << arr[i] << " ";  }  cout << endl;  return 0;}

Time and Space Complexity of Bubble Sort

We are using two nested loop for sorting list using bubble sort so Time Complexity of Bubble Sort will be O(n^2).

Since we are not using any additional space in Bubble Sort so space complexity will be O(1).

Advantages of Bubble Sort

  • Bubble Sort is easy to understand and implement, making it a good choice for educational purposes or small datasets.
  • Bubble Sort sorts the list in place, requiring no additional memory for temporary data structures.

Disadvantages of Bubble Sort

  • Bubble Sort has a time complexity of O(n^2), making it highly inefficient for large datasets.
  • Due to its slow performance, Bubble Sort is rarely used in real-world applications where quick sorting is essential.

Conclusion

Bubble Sort is a simple and easy-to-understand sorting algorithm, but it is far from efficient for sorting large datasets. While it may not be the best choice for many real-world applications, it serves as a valuable learning tool for those new to programming and algorithms.

Understanding Bubble Sort's principles can pave the way for exploring more efficient sorting algorithms like Quick Sort, Merge Sort, or Heap Sort as your programming skills advance.
