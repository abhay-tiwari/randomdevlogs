Min Steps in Infinite Grid | Interview Coding Problem | JavaScript

You are in a limitless 2D grid, where you can move in any of the eight directions. Each movement allows you to move from your current point (x, y) to adjacent points such as (x-1, y-1), (x-1, y), (x-1, y+1), (x, y-1), (x, y+1), (x+1, y-1), (x+1, y), and (x+1, y+1). Now, You are given sequence of points. The challenge is to determine the minimum number of steps required to reach each point in the given order.

Problem can broken down in following points

  • Given a sequence of points.
  • Start from the first point and move to successive points.
  • Objective: Find the shortest path to reach each point in the provided order.
  • Minimize the number of steps taken.
  • Consider eight possible directions for movement.
