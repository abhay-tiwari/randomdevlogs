What is Linear Search?

Linear Search is a basic search algorithm that checks each element in a list until match of element is found or entire list in traversed. It is easy to implement and works on both sorted and unsorted data.

How Does Linear Search Works?

Step by Step breakdown of Linear Search Alogrithm is as follows.

  1. Start at the first element of the list.
  2. Compare the current element with the target value.
  3. If the current element is equal to the target value, return the index of the element.
  4. If the current element is not equal to the target value, move to the next element in the list.
  5. Repeat steps 2-4 until a match is found or the end of the list is reached.
  6. If the end of the list is reached without finding a match, element is not present in the list.

Implementation of Linear Search in C++

Implementation of Linear Search in C++ is as follows.

          #include<iostream>using namespace std;int linearSearch(int arr[], int n, int x) {  for(int i=0; i< n; i++) {    // if element is found, return the index.    if(arr[i] == x) {      return i;     }  }  // return -1 if array is traversed and element is not found.  return -1;}int main() {  int arr[] = {4, 2, 5, 1, 6, 7};  int size = sizeof(arr)/sizeof(arr[0]);  cout << linearSearch(arr, size, 1) << endl; // output = 3}

Advantages of Linear Search

  1. Simplicity: Linear search is easy to understand and implement, making it a valuable choice for small datasets or when simplicity is prioritized.
  2. Works with Unordered Data: Linear search can be used on unordered data, as it doesn't rely on any specific order of elements in the list.
  3. Low Memory Usage: It has a small memory footprint since it doesn't require additional data structures or memory allocation.

Limitations of Linear Search

  1. Inefficient for Large Datasets: Linear search has a time complexity of O(n), meaning its performance decreases as the size of the dataset grows. For large datasets, more efficient algorithms like binary search are preferred.
  2. Not Suitable for Sorted Data: When data is sorted, linear search is highly inefficient compared to binary search, which can take advantage of the ordered structure.

Real-World Applications

  1. Phonebook Search: Linear search can be used to find a contact in a phonebook. It sequentially checks each entry until it finds the desired contact.
  2. To-Do List Search: When searching for a specific task in a to-do list, linear search can be employed to scan through the list and locate the task.
  3. Text Editors: Some text editors use linear search to find and highlight instances of a particular word or phrase within a document.

Conclusion

The Linear Search algorithm is fundamental method for finding elements in a list or array. While it has limitations such as performance issues for large datasets, it remains useful tool in various real-world applications where simplicity and ease of implementation are key priority. Understaging linear search provides a solid foundation for exploring more complex searching techniques in the field of computer science and algorithms
