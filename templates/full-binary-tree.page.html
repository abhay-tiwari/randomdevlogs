What is Full Binary Tree?

Full Binary tree is an important type of binary tree. They have a unique properties that makes them different from other type of binary trees and these properties make useful in various applications in computer science. In this blog post, we will explore fundamentals of full binary trees, their characteristics and their practical uses.

A Full Binary Tree is special type of binary tree where each node has zero or two children i.e. Every level of the tree is fully occupied and there are no nodes with only one child. The structure of full binary tree ensures a balanced distribution of nodes.

C++ Code to Check if a Binary Tree is Full Binary Tree

C++ code to check if a Binary Tree is Full Binary Tree is as follows.

   #include<iostream>using namespace std;class TreeNode {public:  int val;  TreeNode *left;  TreeNode *right;  TreeNode(int val) {    this -> val = val;  }};bool isFullBinaryTree(TreeNode* root) {  // return true if current node is NULL.  if(root == NULL) {    return true;  }  // return true if node is leaf node i.e. left and right child nodes are not present.  if(root -> left == NULL && root -> right == NULL) {    return true;  }  // if both left and right child nodes are present,  // recusively call isFullBinaryTree function for both nodes.  if(root -> left != NULL && root -> right != NULL) {    return isFullBinaryTree(root -> left) && isFullBinaryTree(root -> right);  }  // return false if above conditions are not met.  return false;}int main() {  TreeNode *root = new TreeNode(1);  TreeNode *two = new TreeNode(2);  TreeNode *three = new TreeNode(3);  TreeNode *four = new TreeNode(4);  TreeNode *five = new TreeNode(5);    root -> left = two;  root -> right = three;  two -> left = four;  two -> right = five;  if(isFullBinaryTree(root) == true) {    cout << "Binary Tree is Full Binary Tree" << endl;   }  else {    cout << "Binary Tree is not Full Binary Tree" << endl;  }  return 0;}

Key Characteristics of Full Binary Tree

Key Characteristics of Full Binary are as follows

1. Balanced Structure

Full binary trees are inherently balanced because each level of the tree is as full as possible. This balance makes them suitable for various applications that require efficient searching and traversal.

2. Node Count

In a full binary tree, the number of nodes on each level doubles as you move down the tree. For example, the root level has 1 node, the next level has 2 nodes, the level after that has 4 nodes, and so on. This doubling of nodes is a key characteristic of full binary trees.

3. Height

The height of a full binary tree with 'n' nodes is 'log2(n+1)'. This logarithmic height is a significant advantage in terms of efficiency for operations such as searching, insertion, and deletion.
