Preorder Travesal of Binary Tree using Recursion

Binary Tree is a fundamental data structure in computer science and used to store and get data efficiently. Tree Traversal is an important operation while working with a Binary Tree.

There are following three methods of Tree Traversal.

  • Preorder Traversal
  • Inorder Traversal
  • Postorder Traversal

In this blog, we will explore preorder traversal of a Binary Tree.

How Preorder Traversal Works

Step by Step breakdown of Preorder Traversal using recursion is as follows.

  • If current is not null, visit current node.
  • Recursively visit left subtree of current node.
  • Recursively visit right subtree of current node.

Preorder Traversal of Binary Tree in C++

C++ implementation of Preorder Traversal using recursion is as follows.

      #include<iostream>using namespace std;// class to create a tree nodeclass TreeNode {public:  int data;  TreeNode *left;  TreeNode *right;  TreeNode(int data) {    this->data = data;    left = NULL;    right = NULL;  }};void preorderTraversal(TreeNode *node) {  // return if current node is NULL  if(node == NULL) {    return;  }  // print data of current node.  cout << node -> data << " ";  // recursively traverse left subtree of current node.  preorderTraversal(node -> left);  // recursively traverse right subtree of current node.  preorderTraversal(node -> right);}int main() {  // code to create a tree same as given example  TreeNode *root = new TreeNode(1);  TreeNode *two = new TreeNode(2);  TreeNode *three = new TreeNode(3);  TreeNode *four = new TreeNode(4);  TreeNode *five = new TreeNode(5);  TreeNode *six = new TreeNode(6);  TreeNode *seven = new TreeNode(7);  root -> left = two;  root -> right = three;  two -> left = four;  two -> right = five;  three -> left = six;  three -> right = seven;    preorderTraversal(root);}

Time and Space Complexity of Preorder Traversal

The time complexity of the recursive preorder traversal is O(n), where n is the number of nodes in the binary tree. This is because we visit each node once and perform constant time operations at each node.

Space Complexity of the recursion preorder traversal is O(1) since we are not using any additional space.
