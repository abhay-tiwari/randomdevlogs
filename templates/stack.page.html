What is Stack?

Stack is fundamental data structure in computer science and plays important role in various applications and algorithms. Stack is a linear data structure that follows the Last-In-First-Out(LIFO) principle i.e. the last element added to the stack is the first one to be removed. In this blog, we will explore

  • Basics of Stacks
  • Various Stack Operations
  • Real World Applications of Stack

How Stack Works?

At its core stack is a collection of elements with folowing operations.

  1. Push: Push adds an element to the top of stack.
  2. Pop: Pop removes an element from the top of the stack.
  3. Peak: Peak returns element from the top of stack if stack is not empty.
  4. isEmpty: isEmpty returns true if stack has no elements else it returns false.

Stacks can be implemented using various data structures like arrays or linkedlist depending upon the requirement of the application.

Key Features of Stack

  1. LIFO Principle: As Mentioned earlier, stack follows the Last-In-First-Out principle. Last element pushed onto stack is the first one to be popped off.
  2. Single Access Point: Stack has only one acceess point. Elements can be added or removed only from top of the stack. This feature makes stack useful for specific cases.

Stack Implementation in C++

In this blog post, we will using class and array to implement stack in C++.

We will define max size of stack globally and create a stack class.

      #include<iostream>using namespace std;// max size of stackconst int max_size = 6;class stack {private:  int stack[max_size];  // top of the stack.  int top;public:  stack() {    top = -1;  }};

Push Operation

Push Operation add an element on top of the stack if stack is not full. After adding push method to Stack class C++ code looks as follows:

Pop Operation

Pop operation removed and returns element of top of the stack if stack is not empty. Stack class after adding pop method to class is as follows.

    #include<iostream>using namespace std;// max size of stackconst int MAX_SIZE = 6;class Stack {private:  int stack[MAX_SIZE];  // top of the stack.  int top;public:  Stack() {    top = -1;  }  // method to check if stack is empty.  bool isEmpty() {    return top == -1;  }  // metho to check if stack is full.  bool isFull() {    return top == MAX_SIZE - 1;  }  // method to push element in stack.  void push(int element) {    if(isFull() == true) {      cout << "Stack is full." << endl;      return;    }    top++;    stack[top] = element;  }  // method to pop top element of stack.  int pop() {    if(isEmpty() == true) {      cout << "Stack is empty" << endl;      return -1;    }    int x = stack[top];    top--;        return x;  }};

Real World Applications of Stack

Stack has following real-world applications.

  1. Function Call Stack: In programming languages such as C++ and Java, a stack is used to track of function calls and their local variables. When a function is called, its information is pushed into stack and when it returns, function is popped off.
  2. Expression Evalution: Stacks are used to evaluate mathematical expression such as expressions written as infix notation. Stacks help maintain the correct order of operations.
  3. Undo and Redo Functionality: Many applications such as text editor and graphics design apps use stack to implement undo and redo functionality. Each action pushed to stack allowing user to backtrack and redo operations.
  4. BackTracking Algorithms: Algorithms like depth first search (DFS) use stack to keep track of visible nodes and explore path in systematic manner.

Conclusion

Stacks are fundamental data structure with simple yet powerful concept. Their Last-In-First-Out (LIFO) behaviour and limited access make them well-suited for various use cases of computer science and application development. Understanding stacks is essential for any programmer or computer scientist as stacks are building block for many advanced data structure and algorithms.
