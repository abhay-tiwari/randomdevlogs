Merge Two Binary Trees | Leetcode # 617

In this blog post, we're going to solve interview coding problem: merging two binary trees. You are given two binary trees, root1 and root2, and your task is to create a new binary tree by merging them according to following rules.

  • When two nodes overlap, you should sum their values to create a new node in the merged tree.
  • If a node exists in one tree but not the other, the non-null node should be used as is.

Solution

We will traverse both trees if there is a overlap, will add both node values. Solution in C++ as follows.

      /** * Definition for a binary tree node. * struct TreeNode { *     int val; *     TreeNode *left; *     TreeNode *right; *     TreeNode() : val(0), left(nullptr), right(nullptr) {} *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {} *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {} * }; */  class Solution {public:    TreeNode* mergeTrees(TreeNode* root1, TreeNode* root2) {    if(root1 == NULL && root2 == NULL) {      return root1;    }    if(root2 == NULL) {      return root1;    }    if(root1 == NULL) {      return root2;    }    root1 -> val = root1 -> val + root2 -> val;    root1 -> left = mergeTrees(root1 -> left, root2 -> left);        root1 -> right = mergeTrees(root1 -> right, root2 -> right);    return root1;    }};

Code Explanation

Explanation of above code is as follows.

  1. If both root1 and root2 are NULL, it returns NULL. This is the base case and signifies that there's no node to merge.
  2. If root1 is NULL, it returns root2. This means the root2 tree continues as it is.
  3. If root2 is NULL, it returns root1. This signifies that the root1 tree continues as it is.
  4. If both root1 and root2 exist, it sums their values and assigns the result to root1.
  5. It recursively merges the left subtrees and right subtrees by calling mergeTrees on root1->left and root2->left, as well as root1->right and root2->right.
  6. The method returns the merged root1 tree.

Merging two binary trees is a common problem encountered in tree-related algorithms. The provided C++ solution is an efficient way to merge two trees while summing overlapping node values. You can adapt this code to handle more complex binary trees and explore other tree-related problems.
