Root Equals Sum of Children | Leetcode # 2236

In this blog, we solve interview coding problem Root Equals Sum of Children. We are given the root of Binary Tree which have exactly three nodes (root, left child node, right child node). Our goal is check if sum of left and right nodes is equal to root node value or not.

Function should return true if sum of left and right child nodes is equal to root root else return false.

Example 1

10
4
6

In above case, function should return true since sum of child nodes (4 + 6 = 10) is equal to root node 10.

Example 2

10
4
3

In above case, function should return false since sum of child nodes (4 + 3 = 7) is not equal to root node 10.

Solution

Solution of above problem using C++ is as follows.

        /** * Definition for a binary tree node. * struct TreeNode { *     int val; *     TreeNode *left; *     TreeNode *right; *     TreeNode() : val(0), left(nullptr), right(nullptr) {} *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {} *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {} * }; */class Solution {public:  bool checkTree(TreeNode* root) {        int rootValue = root -> val;        int leftNodeValue = root -> left -> val;    int rightNodeValue = root -> right -> val;    if(rootValue == leftNodeValue + rightNodeValue) {      return true;    }    else {      return false;    }  }};

Code Explanation

In the checkTree method, we start by extracting the values of the root and its left and right children. Then, we compare whether the sum of the left and right child values is equal to the root's value. If it is, the method returns true, indicating that the tree is a Sum Tree; otherwise, it returns false.
